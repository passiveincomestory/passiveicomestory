Theme Description:

My Portfolio is a multi-page starter template for creatives. The homepage lists all the portfolio items. The about page lists a bio with a chronological display of skills and successes. The contact page features a large map and a contact form.

Theme Features:

  • Portfolio starter website
  • 3 pages: Home, About, Contact
