Theme Description:

Blog With Sidebar is a two-column blog page template. Each blog post has a blog post heading, a date, an image, and a description block. There is also a sidebar on the right side. This template is ideal for simple and minimalist blogs.

Theme Features:

  • Single column blog layout
  • Blog post heading, image, and additional information areas
  • Placeholder images by Placehold.it
